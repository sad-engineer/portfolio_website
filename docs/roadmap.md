# Дорожная карта проекта

Обновлено: 25.11.2025

## Этап 1. Сайт-визитка
- Перенести шаблон StartBootstrap в FastAPI + Jinja.
- Настроить статический хостинг и локальный dev-сервер.
- Подготовить контент: био, навыки, контакты, проекты-заглушки.
- Добавить базовые автотесты и линтеры.

## Этап 2. Проекты с базами данных
- Создать три независимых сервиса (FastAPI) с подключением к БД.
- Настроить миграции, конфигурацию окружений и тестовые данные.
- Добавить описание и ссылки в сайт-визитку.

## Этап 3. Сервисы над БД
- Выделить слой приложений/сервисов, работающих с ранее созданными БД.
- Определить API-контракты, протестировать бизнес-логику.
- Обновить визитку с описанием и диаграммами взаимодействия.

## Этап 4. Верхнеуровневое приложение
- Создать фронтенд или API-шлюз, агрегирующий результаты сервисов.
- Настроить авторизацию, кэширование и оркестрацию запросов.
- Добавить интеграционные тесты и мониторинг.

## Этап 5. Независимые проекты
- Поддерживать каталог sandbox (игры, эксперименты).
- Интегрировать их в визитку как отдельные разделы/демо.
- При необходимости разворачивать в отдельные сервисы.

## Этап 6. Непрерывное развитие
- Добавлять новые проекты по мере появления идей.
- Поддерживать документацию (`docs/`), обновлять дорожную карту.
- Развивать инфраструктуру (CI/CD, контейнеризация, деплой).
